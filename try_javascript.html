üí° Tips dan Best Practices

  • Selalu pastikan kondisi loop akan berubah - Jangan sampai terjadi infinite loop
  • Inisialisasi variabel dengan benar - Pastikan variabel counter sudah memiliki nilai awal
  • Update variabel di dalam loop - Jangan lupa increment/decrement atau mengubah kondisi
  • Gunakan while untuk kondisi yang tidak diketahui jumlah iterasinya
  • Pertimbangkan menggunakan for loop jika jumlah iterasi sudah diketahui
